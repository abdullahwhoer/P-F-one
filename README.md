### Cipherz – PORTFOLIO Website

A modern, responsive digital agency website built using HTML, CSS, JavaScript, and Bootstrap. This project showcases a professional landing page suitable for agencies, startups, freelancers, and technology businesses.

## 🚀 Live Preview

Add your live website link here:

**Link:** https://cipherz-portfolio.vercel.app/

## 📌 Features

* Fully Responsive Design
* Modern and Clean UI
* Hero Section
* About Section
* Services Section
* Portfolio Showcase
* Team Section
* Testimonials
* Pricing Plans
* FAQ Section
* Contact Form
* Smooth Scrolling
* Bootstrap Components
* Cross-Browser Compatible
* Easy to Customize

## 📥 Installation

1. Clone the repository

```bash
git clone https://github.com/abdullahwhoer/P-F-one.git
```

2. Open the project folder.

3. Launch `index.html` in your browser.

No additional installation is required.
